HTML to Image CLI

HTML을 한 장의 이미지로 완성 이미지로 바꾸는 CLI

최신 버전 다운로드 GitHub 보기

Windows, macOS, Linux용 패키지를 릴리즈에서 바로 받을 수 있습니다.

Latest release

v1.2.0

최신 릴리즈 정보를 불러오는 중입니다.

./markup-img example.html output.png

cat page.html | ./markup-img - banner.jpg

./markup-img page.html - > result.png

가벼운 배포

Neutralinojs 기반으로 필요한 구성만 묶어 전달하므로 배포 패키지를 작게 유지할 수 있습니다.

파이프라인 친화적

파일 입력과 stdin, stdout 흐름을 모두 지원해 CI나 다른 CLI 도구와 바로 연결할 수 있습니다.

멀티 플랫폼

Windows, macOS, Linux 릴리즈를 제공해 같은 워크플로를 여러 환경에서 재사용할 수 있습니다.

세 단계로 바로 적용

1

릴리즈 다운로드

운영체제에 맞는 압축 파일을 받아 풀면 즉시 실행할 수 있습니다.

2

HTML 준비

전체 문서를 캡처하거나 `#markup-img` 요소 하나만 지정해 필요한 영역만 출력할 수 있습니다.

3

이미지 생성

PNG나 JPEG로 저장하거나 stdout으로 흘려 후속 이미지 처리 파이프라인에 붙일 수 있습니다.
