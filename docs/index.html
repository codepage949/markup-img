lightweight

가벼운 배포

Neutralinojs 기반으로 필요한 구성만 묶어 전달하므로 배포 패키지를 작게 유지할 수 있습니다.

pipeline

파이프라인 친화적

파일 입력과 stdin, stdout 흐름을 모두 지원해 CI나 다른 CLI 도구와 바로 연결할 수 있습니다.

cross-platform

멀티 플랫폼

Windows, macOS, Linux 릴리즈를 제공해 같은 워크플로를 여러 환경에서 재사용할 수 있습니다.
